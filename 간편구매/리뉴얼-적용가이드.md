# 🎉 간편구매 버튼 영역 - 리뉴얼 적용 가이드

고도화된 구매 버튼 디자인이 적용된 리뉴얼 파일입니다.

---

## 📁 생성된 파일

```
간편구매/
├── 간편구매-리뉴얼.html     ← 리뉴얼된 HTML
├── 간편구매-리뉴얼.css      ← 리뉴얼된 CSS (고도화 스타일)
├── 간편구매-리뉴얼.js       ← 기존 JS (변경 없음)
└── 리뉴얼-적용가이드.md     ← 이 파일
```

---

## ✨ 주요 변경사항

### 1. 버튼 구조 변경

**이전 (4개 버튼 2x2 그리드)**
```
┌─────────────┬─────────────┐
│ 장바구니담기 │ 바로구매하기 │
├─────────────┼─────────────┤
│  간편구매   │ N구매하기   │
└─────────────┴─────────────┘
```

**이후 (4개 버튼 2+2 구조)**
```
기본 구매 버튼
┌─────────────┬─────────────┐
│  장바구니   │  바로구매   │
└─────────────┴─────────────┘

간편결제 버튼
┌─────────────┬─────────────┐
│  네이버페이  │  카카오페이  │
└─────────────┴─────────────┘
```

### 2. 디자인 고도화

| 항목 | 개선 내용 |
|------|----------|
| **그라데이션** | 바로구매, 네이버페이 버튼에 그라데이션 적용 |
| **그림자** | 깊이감 있는 그림자 효과 |
| **호버 효과** | 상승 애니메이션 + 그림자 강화 |
| **클릭 효과** | 눌림 효과 (scale 0.98) |
| **포커스** | 접근성 향상 (아웃라인 표시) |
| **반응형** | 3단계 브레이크포인트 |

### 3. 기능 유지

✅ 메이크샵 장바구니/바로구매 기능 100% 유지
✅ 메이크샵 네이버페이 스크립트 연동
✅ 메이크샵 카카오페이 스크립트 연동
✅ 기존 JavaScript 로직 100% 호환

## 🚀 적용 방법

### 방법 1: 전체 파일 교체 (권장)

```bash
# 백업 (혹시 모를 문제 대비)
cp 간편구매.html 간편구매-백업.html
cp 간편구매.css 간편구매-백업.css

# 리뉴얼 파일로 교체
mv 간편구매-리뉴얼.html 간편구매.html
mv 간편구매-리뉴얼.css 간편구매.css
```

### 방법 2: CSS만 교체

기존 CSS 파일(`간편구매.css`)에서 아래 부분을 찾아 삭제:

```css
/* 169-177 라인 */
/* 구매 탭 2x2 그리드 (장바구니 담기, 바로 구매하기, 간편구매, N 구매하기) */
.goods--btns-purchase-grid { ... }
.goods--btns-purchase-grid .btn-purchase { ... }
.goods--btns-purchase-grid .btn-cart { ... }
.goods--btns-purchase-grid .btn-buy-now { ... }
.goods--btns-purchase-grid .btn-easy-purchase { ... }
.goods--btns-purchase-grid .btn-naver-pay { ... }
```

그리고 `간편구매-리뉴얼.css` 내용을 그 위치에 붙여넣기.

## 📱 반응형 지원

### PC (992px 이상)
- 버튼 높이: 56px
- 폰트 크기: 16px
- 2열 가로 배치

### 태블릿 (768px ~ 991px)
- 버튼 높이: 54px
- 폰트 크기: 15px
- 2열 가로 배치

### 모바일 (767px 이하)
- 버튼 높이: 54px → 52px (480px 이하)
- 폰트 크기: 15px → 14px (480px 이하)
- **1열 세로 배치**

## ⚠️ 주의사항

### 1. 메이크샵 스크립트 호환성

```html
<!-- 이 부분은 메이크샵이 자동으로 생성하므로 수정 불필요 -->
<div id="nhn_btn">
    <!--/btn_nhn/-->
</div>

<div id="kakaopay_order_btn">
    <!--/btn_kakaopay/-->
</div>
```

### 2. CSS 우선순위

`!important` 사용으로 메이크샵 기본 스타일을 오버라이드합니다.
메이크샵 업데이트 시에도 스타일이 유지되지만, 충돌 발생 시 확인 필요.

### 3. 브라우저 호환성

- `gap` 속성: IE11 미지원 (필요 시 margin 대체)
- `cubic-bezier`: 모든 모던 브라우저 지원
- `linear-gradient`: 모든 모던 브라우저 지원

## 🐛 문제 해결

### Q1. 버튼이 세로로만 표시됩니다
**A.** 브라우저 창 크기를 확인하세요. 767px 이하에서는 의도적으로 세로 배치됩니다.

### Q2. 네이버페이/카카오페이 버튼이 표시되지 않습니다
**A.** 메이크샵 관리자에서 해당 간편결제를 활성화했는지 확인하세요.

### Q3. 호버 효과가 작동하지 않습니다
**A.** 모바일에서는 호버 효과가 없습니다. PC 브라우저에서 확인하세요.

### Q4. 기존 디자인으로 되돌리고 싶습니다
**A.** 백업 파일을 복원하세요:
```bash
mv 간편구매-백업.html 간편구매.html
mv 간편구매-백업.css 간편구매.css
```
